# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, active states, key accents
- **Secondary:** `amber` — Used for badges, warning states, attention indicators
- **Neutral:** `slate` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-blue-600 hover:bg-blue-700 text-white`
Secondary badge: `bg-amber-100 text-amber-800 dark:bg-amber-900/40 dark:text-amber-300`
Neutral text: `text-slate-600 dark:text-slate-400`
Card border: `border-slate-200 dark:border-slate-700`
Active nav item: `bg-blue-50 text-blue-700 dark:bg-blue-950/40 dark:text-blue-300`
